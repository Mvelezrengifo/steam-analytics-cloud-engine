# Steam Data Lakehouse & Dashboard 🚀
Architecture:
Steam Dataset → BigQuery (Lakehouse Layers) → Django Backend → Web Dashboard

Proyecto de Ingeniería de Datos y Desarrollo Web que integra un flujo de datos desde un Lakehouse en Google Cloud (BigQuery) hasta un Dashboard dinámico en Django.

## 🛠️ Tecnologías
- **Backend:** Django (Python)
- **Cloud:** Google BigQuery (GCP)
- **Data:** SQL, ETL Pipelines
- **Architecture:** Lakehouse (Medallion Architecture)

## 📊 Características
- Conexión en tiempo real con BigQuery mediante Service Accounts.
- Arquitectura de carpetas profesional y escalable.
- Visualización de métricas de popularidad y reseñas de Steam.
